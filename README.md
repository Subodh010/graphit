> A graphing calculator

## Installation

### Dependencies

- C/C++ compiler
- SFML

### Build from source

Linux:
```
git clone https://github.com/soupfoo/graphit.git
cd graphit
sh run.sh
```

Windows:
```
git clone https://github.com/soupfoo/graphit.git
cd graphit
./run.ps1
```
